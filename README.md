# Jedi Adventure

A simple adventure game built using Python.

## About

In this game, you play as a Jedi exploring a hidden galaxy. Your goal is to collect all the lightsaber parts and defeat the Sith Lords.

The game board is randomly generated, so each playthrough is different.

## How to Run

Make sure Python is installed, then run:

```python
python jedi_adventure.py
```

## Controls

- `u` - Move Up
- `d` - Move Down
- `l` - Move Left
- `r` - Move Right
- `q` - Quit

## Objective

- Explore the board
- Collect all lightsaber parts
- Avoid Sith Lords until your lightsaber is complete
- Defeat all Sith Lords to win
